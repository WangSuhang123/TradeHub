## 2. 改造目标与架构设计

### 2.1 改造前后对比

| 维度 | 改造前（CookieShop） | 改造后（TradeHub） |
|------|---------------------|-------------------|
| 项目主题 | 蛋糕/甜品电商 | 校园二手交易 |
| 前端框架 | Bootstrap 3 + 自定义CSS | Tailwind CSS CDN + 最小自定义 |
| Servlet数量 | 36个（@WebServlet注解） | 5-6个（BaseServlet反射） |

| 订单状态 | 未付款→已付款→已发货→已完成 | 待确认→交易完成→已取消 |
| 支付流程 | 选择微信/支付宝 | 无（线下交易） |
| 图片存储 | goods表cover+image1+image2字段 | product表单图路径+预留扩展 |
| 用户登录 | 用户名/邮箱+密码 | 学号+密码 |

## 4. BaseServlet 统一入口架构

### 4.1 设计原理

将36个独立Servlet合并为5-6个业务Servlet，通过反射机制分发请求：

```
原架构（36个Servlet）：
  /index → IndexServlet.doGet()
  /user_login → UserLoginServlet.doPost()
  /user_register → UserRegisterServlet.doPost()
  /user_logout → UserLogoutServlet.doGet()
  /user_changeaddress → UserChangeAddressServlet.doPost()
  /user_changepwd → UserChangePwd.doPost()
  ...（共36个映射）

新架构（BaseServlet反射）：
  /product?method=list   → ProductServlet.list()
  /product?method=detail → ProductServlet.detail()
  /product?method=publish → ProductServlet.publish()
  /product?method=search → ProductServlet.search()
  /user?method=login     → UserServlet.login()
  /user?method=register  → UserServlet.register()
  /user?method=logout    → UserServlet.logout()
  ...（共5个Servlet处理所有业务）
```

### 4.2 BaseServlet 实现

```java
package servlet;

import javax.servlet.ServletException;
import javax.servlet.http.HttpServlet;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;
import java.io.IOException;
import java.lang.reflect.Method;

public class BaseServlet extends HttpServlet {
    @Override
    protected void service(HttpServletRequest request, HttpServletResponse response) 
            throws ServletException, IOException {
        request.setCharacterEncoding("utf-8");
        response.setContentType("text/html;charset=utf-8");

        String methodName = request.getParameter("method");
        if (methodName == null || methodName.isEmpty()) {
            methodName = "index";
        }

        try {
            Method method = this.getClass().getMethod(methodName, 
                HttpServletRequest.class, HttpServletResponse.class);
            String result = (String) method.invoke(this, request, response);
            
            if (result != null) {
                if (result.startsWith("redirect:")) {
                    response.sendRedirect(request.getContextPath() + result.substring(9));
                } else if (result.startsWith("json:")) {
                    response.getWriter().print(result.substring(5));
                } else {
                    request.getRequestDispatcher(result).forward(request, response);
                }
            }
        } catch (Exception e) {
            e.printStackTrace();
            response.sendRedirect(request.getContextPath() + "/index.jsp");
        }
    }
}
```
